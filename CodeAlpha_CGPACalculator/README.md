CGPA Calculator
A simple C++ application that helps students calculate their CGPA based on grades and credit hours entered for each subject. The program computes grade points, semester GPA, and overall CGPA accurately and efficiently.
Features
Input grades and credit hours
Calculate GPA and CGPA
User-friendly console interface
Fast and accurate results
Technologies Used
C++
Object-Oriented Programming (OOP)
How to Run
Compile the program using a C++ compiler.
Run the executable.
Enter subject grades and credit hours when prompted.
View the calculated GPA/CGPA.
